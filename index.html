User Flow Story

1. Super Admin Setup

  • Secure login to UpBuild platform
  • Create new apartment complexes
  • Define basic settings (visitor rules, maintenance)

2. Assigning Apartment Admin

  • Assign apartment managers
  • Send invite emails for setup

3. Apartment Admin Management

  • Add staff (Receptionists, Guards, Technicians)
  • Configure apartments and tenant assignments

4. Staff Operations

  • Receptionists manage visitors
  • Guards monitor entries
  • Technicians manage maintenance tasks

5. Tenant Interactions

  • Pre-register visitors, raise complaints, submit feedback

6. Visitor Management

  • Pre-register or manual register
  • QR code badges issued
  • Check-in/out times recorded

9. Performance and Reports

10. Platform Expansion

  • Plan subscriptions (Free/Paid) for scaling
